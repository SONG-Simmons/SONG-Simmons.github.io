d3.js——first contact

一.选择器的简洁与强大

二.enter()与exit()

enter()：当数据（）数组长度大于DOM长度，多出来的数据就叫做enter

补足DOM，同时为DOM内填充上数据。
exit()：当数据（）数组长度小于DOM长度，DOM多出来的数据就叫做exit

三.绘制树状图的基本步骤

1.选中页面，给页面添加svg标签，设置svg绘制区域的宽高，添加g标签，给svg定位

var width = 500,
    height = 500;
var svg = d3.select('svg')
    .append('svg')
    .attr('width', width)
    .attr('heiht', height)
    .append('g')
    .attr('tranform', 'translate(50,0)');

这里分为5个步骤，不过由于d3链式操作的简洁性，可以用很短的代码完成。

2.生成树状布局，设置尺寸

var tree = d3.layout.tree()
    .size([width, height-200])

这是d3官方文档中的API，因为其API数量庞大，在需要用时可收手动查阅。

3.对角线生成器

连接数的节点时平滑的曲线叫做对角线；使用projection方法，使得其转动90°，成为一个真正意义上的“树“，其参数d返回一个数组，实现90°转动。

4.请求数据
